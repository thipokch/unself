Hi! We are so —genuinely— glad that you're here, looking to contributing to Unself! 
We greatly appreciate any help and support. 

## How to contribute

Here are some ways you can contribute:

- Report bugs and suggest new features by creating an issue on GitHub
- Help us improve documentation by suggesting changes or writing new documentation
- Contribute to the codebase by creating pull requests
- Test the project and report any issues you find
- Share the project with your friends and colleagues
- Donate to support the development of the project

## Code of Conduct

We want to ensure a welcoming and inclusive environment for all contributors. We have adopted the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/0/code_of_conduct/) to govern interactions within the project's community. By participating, you agree to abide by its terms.
